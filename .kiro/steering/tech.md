# Tech Stack

## Core Technologies
- **HTML5 Canvas** - Game rendering and graphics
- **Vanilla JavaScript (ES6+)** - Game logic and mechanics
- **CSS3** - UI styling and overlays

## Architecture
- No build system or bundler required
- No external dependencies or frameworks
- Pure client-side execution
- Single-page application structure

## Key Libraries & APIs
- Canvas 2D Context API for rendering
- RequestAnimationFrame for game loop
- Keyboard Events API for input handling
- Image API for sprite loading

## Running the Game
Simply open `index.html` in a modern web browser. No build, compile, or installation steps required.

## Development
- Edit files directly and refresh browser to see changes
- Use browser DevTools console for debugging
- Test in Chrome, Firefox, Safari, or Edge
